使用 Github Pages 和 Hexo 搭建个人博客

发表于 2014-07-27   |  

参考了许多网上的教程,终于搭建出属于自己的个人博客,在此记录下遇到的一些问题及流程

缠绕的问题
1. 需不需要建立 github-pages 分支,答案是 不需要
2. 在完成所有的操作后需要手动git push到主分支吗,结果是根本不需要将  
github上的仓库clone到本地,在本地任意文件夹下配置好deploy参数之后,只  
需执行hexo g -d 可自动commit,push到github上(后面会提到一个站点配置文件)
搭建流程

  • 在 github 的操作

    • 注册账号并创建一个名称为 username.github.io 的仓库,此处的username为你注册账号的用户名, 参考地址
    • 通过 SSH Key 与本地建立 git 链接 Generating an SSH key
    • git的学习资料

  • 本地电脑端的操作

    • 安装 HEXO 参考 hexo文档 依次按顺序来完成即可完成建站, Node.js 和 git ,再使用终端命令安装 hexo

      $ npm install -g hexo-cli

    • 在命令行Blog目录下clone主题到themes目录下

      git clone https://github.com/iissnan/hexo-theme-next.git themes/NEXT
